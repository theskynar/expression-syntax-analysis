# syntax-analysis

This application is a simples syntax expression analyzer.

## Examples: 

### Expression 1: 
  ```
    2 * 5 * 6 will be accepted
  ```
### Expression 2: 
  ```
    2 * (5 * (3 + 20) - 4) + 9 will be accepted
  ```

### Expression 3: 
  ```
    2 * (5 * (3 + 20) - 4 + 9 will be rejected
  ```
